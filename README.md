# Smart Care Sense (HealthPulse)

![HealthPulse Banner](https://images.unsplash.com/photo-1576091160550-2173dba999ef?auto=format&fit=crop&q=80&w=1200&h=400)

**Smart Care Sense** is a modern, real-time health monitoring platform that integrates wearable technology with AI-driven insights to provide a premium wellness experience.

## ✨ Core Highlights
- 💓 **Live Vitals Tracking**: Real-time heart rate, blood pressure, and activity monitoring.
- 🤖 **AI Health Assistant**: Personalized wellness advice based on your health data.
- ⌚ **Seamless Connectivity**: Support for Bluetooth LE wearables and custom Wi-Fi IoT devices.
- 📊 **Dynamic Visualizations**: Interactive charts and live ECG wave monitoring.
- 🎨 **Premium UI**: Modern glassmorphism design built with Tailwind CSS and Framer Motion.

## 🛠️ Tech Highlights
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Shadcn UI.
- **Backend**: Supabase (PostgreSQL, Auth, Edge Functions).
- **Hardware**: Web Bluetooth API, RESTful Ingestion.
